## جزئیات اعتبارسنجی‌های پیاده‌سازی شده

### اعتبارسنجی شماره تلفن همراه ایرانی
- الگو: `^09\d{9}$`
- باید با 09 شروع شود
- باید دقیقاً 11 رقم باشد
- فقط اعداد مجاز هستند

### اعتبارسنجی نام کاربری
- حداقل 3 کاراکتر، حداکثر 50 کاراکتر
- فقط حروف، اعداد و underscore مجاز است
- تبدیل خودکار به حروف کوچک

### اعتبارسنجی ایمیل
- استفاده از `EmailStr` از Pydantic
- اعتبارسنجی فرمت استاندارد ایمیل

### اعتبارسنجی مختصات جغرافیایی
- عرض جغرافیایی: -90 تا 90
- طول جغرافیایی: -180 تا 180

### Auto-correction در Pagination
- page < 1 → اصلاح به 1
- per_page < 1 → اصلاح به 1
- per_page > 100 → اصلاح به 100
